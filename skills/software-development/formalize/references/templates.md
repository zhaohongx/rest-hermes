# formalize 规格书模板集

> **定位**：L1/L2/L3 三级渐进式模板 + B1-STEM 段。主 SKILL.md 按复杂度级别引用对应模板。

---

## 一、L1 微型模板（mini）

适用：单对象、单输入输出、无规则分支。目标长度 300-600 字。

```
# 形式化规格（微型）

**对象**: <一句话描述>
**输入 → 输出**: <X> → <Y>
**核心规则**: <一条规则，IF-THEN 形式>
**验证**:
- 正例: <input → output>
- 反例: <input → 应拒绝>

**置信度**: 高/中/低
**理由**: <一句话>
```

---

适用：2-5 个对象，含规则分支，无系统级关键词。目标长度 800-1800 字。

```

- **<对象名>**: <属性列表>

[STEM 题在此插入 B1-STEM 段]

### B3. 输出
```
