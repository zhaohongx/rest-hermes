# 模板共享片段

> **定位**：跨 L1/L2/L3/stem 模板的公共结构元素。被各模板按需引用，避免四份重复。
> **使用方式**：模板中 `@include: _shared.md#section-name` 或直接引用对应章节。

---

## 通用置信度段

```
**置信度**: 高 / 中 / 低
**理由**: <一句话>
**降低不确定性的下一步**: <最值得追问的 1 个点>
```

## 通用验证段

```
**验证**:
- 正例: <input → output>
- 反例: <input → 应拒绝>
- 边界例: <极端情况 → 预期行为>
```

## 通用假设声明段

```
### 已显式化的隐含假设
- ✓ <假设>

### 需要确认的不确定项
- ⚠️ <项>

### 用户未提及但建议补充
- <建议>
```

## 状态符号约定

在所有模板中使用统一符号：`✓`（已确认）/ `⚠️`（不确定）/ `✗`（缺失）。不可替代为 ✅/❌/❗。
