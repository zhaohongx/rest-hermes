# STEM-Eng 子模板（工艺/制造/化工类）

> **定位**：补充 B1-STEM 对工程领域覆盖不足的问题。当需求涉及工艺设计、制造流程、化工过程时使用此模板。
> **触发关键词**：工艺、流程、制造、化工、选矿、冶炼、反应器、物料平衡、能量平衡、产率、收率、批次
> **证据来源**：白云石选矿降硅案例——纯 STEM 模板的"坐标系"字段冗余，而"工艺流程图"完全缺失。

---

## 触发判定

满足以下**任一条件**即触发此子模板：
- 需求涉及物理/化学加工过程
- 需求涉及产率/收率/品位等工程指标
- 需求涉及物料流/能量流
- 需求明确为工业场景（非纯理论计算）

**与 B1-STEM 的关系**：STEM-Eng 是 B1-STEM 的**子模板**——先填充 B1-STEM 的变量表/假设 ⚠️，再追加以下工程专属段。

## 使用说明

1. **先用 B1-STEM 的变量表建立数学模型**（符号→含义→单位→取值范围）
2. **再用 E1-E6 追加工程专属段**（B1-STEM 中的"坐标系"字段在工程场景下可替换为"工艺流程图"）
3. **E5（失效模式）不可跳过**——工业场景的失效后果远重于纯理论场景
4. **所有工艺参数标 ⚠️** 除非有实验数据支撑
