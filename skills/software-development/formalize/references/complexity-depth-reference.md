# 复杂度深度参考（8 层统一公理）

> **定位**：扩展 S2 复杂度分级的判定依据。L1-L3 对应第 1-5 层，L3+ 扩展到第 6-8 层。

---

## 一、8 层横向递进

| 层 | 特征 | 判定词 | 典型问题 |
|----|------|--------|---------|
| 1 | 静态·有限·简单 | 单个、固定、不变 | 1+1, 质点匀速运动 |
| 2 | 对立·负向·对称 | 缺少、反向、缺陷 | 负数、空集、摩擦力 |
| 3 | 分割·比例·分形 | 中间态、分数、比例 | 有理数、黄金分割、热传导 |
| 4 | 连续·无限·非线性 | 极限、无穷、混沌 | 实数、湍流、极限运算 |
| 5 | 动态·波动·叠加 | 波、相位、干涉 | 复数、量子态、波动方程 |
| 6 | 旋转·弯曲·扭曲 | 曲率、自旋、非欧 | 四元数、弯曲时空、陀螺仪 |
| 7 | 极限·奇异·不可解 | 奇点、发散、混沌边界 | 超越数、黑洞、NP 完全 |
| 8 | 多维·耦合·全域统一 | 多场耦合、张量、统一 | 统一场论、多元分析、全栈系统 |

---

## 二、4 条纵向进化链

### 2.1 数系进化
```
自然数 → 整数 → 有理数 → 实数 → 复数 → 四元数 → 超越数 → 张量
```

### 2.2 维度进化
```
正整数维 → 负维 → 有理分数维 → 无理实维 → 复维 → 四元数维 → 超越数维 → 张量维
```

### 2.3 代数进化
```
半群 → 群 → 环 → 实数域 → 复数域 → 超复数 → 超越域 → 张量代数
```

### 2.4 物理进化
```
质点力学 → 刚体缺陷 → 分形热力学 → 混沌 → 量子波动 → 弯曲时空 → 奇点黑洞 → 统一场论
```

---

## 三、闭环本质

```
第 8 层的张量/统一场，反向包含前面所有层级：

  整数维  ⊂  张量维（均匀特例）
  实数    ⊂  复数（虚部=0 特例）
  平直空间 ⊂  弯曲空间（曲率=0 特例）
  质点    ⊂  刚体（形状=0 特例）
```

**低级结构是高级结构的特例。**

---

## 四、formalize S2 分级映射

| formalize 级别 | 对应层 | 模板 |
|---------------|--------|------|
| L1 微型 | 第 1 层 | 单对象·单规则 |
| L2 标准 | 第 2-3 层 | 多对象·含规则·有分支 |
| L3 扩展 | 第 4-5 层 | 系统级·多模块·非线性 |
| L3+ | 第 6-8 层 | 多层耦合·统一系统（使用本参考扩展） |

### L3+ 触发条件
- 需求涉及 2+ 条进化链交叉（如数系+物理）
- 需求涉及第 6-8 层特征（曲率/奇点/多场耦合）
- 需求声称"统一"、"全域"、"终极"等闭环关键词

## 五、B1-STEM 如何适配各层

```
第 1-3 层 → 变量: 离散, 方程: 代数式, 假设: ⚠️ 理想化
第 4-5 层 → 变量: 连续, 方程: 微分/积分, 假设: ⚠️ 边界条件
第 6-8 层 → 变量: 张量/多场, 方程: 偏微分/耦合, 假设: ⚠️ 简化场假设
```
